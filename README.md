> 💬 **Max** is an AI-powered chatbot built with **Next.js, TypeScript, and TailwindCSS**.
> It’s designed to deliver intelligent, real-time conversations with a clean, modern, and animated UI. 🚀

---

## ✨ Features

✅ Real-time AI conversations
🎨 Sleek, responsive design (Tailwind + shadcn/ui)
⚡ Powered by Next.js for performance
🧩 Modular & customizable codebase
🎞️ Smooth animations with Framer Motion

---

| Technology            | Purpose                             |

## 🚀 Getting Started

1️⃣ Clone the repository

```bash
git clone https://github.com/abidsk77/max-ai-chatbot.git
```

2️⃣ Navigate to the project folder

```bash
cd max-ai-chatbot
```

3️⃣ Install dependencies

```bash
npm install
```

4️⃣ Run the development server

```bash
npm run dev
```

5️⃣ Open in your browser 🌐
[http://localhost:3000](http://localhost:3000)
